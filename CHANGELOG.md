# Changelog

## 0.1.0-alpha.0

- Added `sideEffects: false` for better bundler compatibility
- GitHub Actions release workflow with npm trusted publishing (OIDC)

## 0.0.1 - Initial experimental release

- `useAgentTool`: register callable tools with state tracking
- `useAgentContext`: expose read-only React state to agents
- `useAgentEvent`: subscribe to tool lifecycle events
- `installPolyfill`: WebMCP polyfill for pre-native browsers
